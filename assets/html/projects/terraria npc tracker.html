Terraria NPC Housing Tracker

A quick tool I made to track NPC housing requirements in Terraria. Specifically to be used while designing worlds and building NPC housing. I myself always wished there were a tool to track which NPCs I've built houses for and which ones I haven't yet. The in-game NPC tracker only shows if they have spawned, not if you've built them a special house. This tool exists to fill that gap.

A school project, it was made with HTML, CSS, and JavaScript. The NPC data is stored in a JSON file, which the JavaScript fetches and parses to generate the checklist on the webpage, depending on the user's selected sources, such as the Calamity or Thorium mod.

Over 85 NPCs across 7 sources can be tracked by this tool. I plan on working on it more in the future, its planned updates are listed at the bottom of the page. Feel free to leave feedback on its GitHub page! [Link]
